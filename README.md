# 📢 Digital Marketing Course Landing Page

A modern, responsive, and user-friendly landing page designed for promoting a Digital Marketing course. This project focuses on clean UI design, structured content, and conversion optimization.


## 🚀 Features

- ✅ Fully responsive design
- ✅ Clean and modern UI
- ✅ Sticky navigation bar
- ✅ Dropdown menu for course sections
- ✅ Hero section with strong CTA (Call-To-Action)
- ✅ Course overview and modules
- ✅ Practical vs Traditional comparison section
- ✅ Curriculum accordion (expandable FAQ style)
- ✅ Pricing plans section
- ✅ Why Choose Us section
- ✅ Smooth scrolling navigation
- ✅ Font Awesome icons integration


## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Roboto)
- Font Awesome Icons


## 📌 Sections Included

- Header (Navigation Bar)
- Hero Section
- Pain Point Section
- Course Overview
- Why Choose Us
- Curriculum (Accordion)
- Pricing Plans
- Reviews (if added)
- Gallery (if added)
- FAQ Section
- Contact Section


## 🎯 Purpose

This landing page is designed to:
- Promote digital marketing courses
- Increase user engagement
- Generate leads and conversions
- Showcase course curriculum and benefits
